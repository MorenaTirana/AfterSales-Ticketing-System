# SESSA MARINE - After Sales Ticketing System

## Tecnologie utilizzate

- HTML
- CSS
- JavaScript
- Node.js
- Express
- MySQL

## Installazione

1. Importare il database

schema.sql

seed.sql

2. Installare i pacchetti

npm install

3. Avviare il server

npm start

4. Aprire

http://localhost:3001
