After Sales Ticketing System

Progetto Universitario - Tecnologie Informatiche per il Web

Descrizione

Sistema web sviluppato per la gestione completa dell'assistenza post-vendita di Sessa Marine. L'applicazione consente la gestione dei ticket di garanzia e ricambi, clienti, dealer, imbarcazioni, assegnazioni ai tecnici, allegati, commenti, storico delle attività e dashboard statistiche.

Tecnologie

  • Node.js
  • Express.js
  • MySQL
  • REST API
  • HTML5
  • CSS3
  • JavaScript ES6

Ruoli

  • Administrator
  • After Sales
  • Technician
  • Customer

Funzionalità

  • Autenticazione con ruoli
  • Gestione utenti
  • Gestione clienti
  • Gestione dealer
  • Gestione imbarcazioni
  • Gestione Ticket in Garanzia
  • Gestione Ticket Ricambi
  • Assegnazione tecnici
  • Storico modifiche ticket
  • Commenti pubblici e interni
  • Gestione allegati
  • Dashboard statistiche
  • Filtri e ricerca ticket

Obiettivo del Progetto

Il progetto è stato sviluppato per simulare un sistema reale di gestione After Sales nel settore nautico, consentendo la gestione centralizzata delle richieste di assistenza, delle garanzie, dei ricambi e delle attività dei tecnici.
