SESSA MARINE

Informazioni sul Progetto

Descrizione

Sistema web sviluppato per la gestione completa dell'assistenza post-vendita di SESSA Marine. L'applicazione permette la gestione di:

  • Ticket Garanzia
  • Ticket Ricambi
  • Clienti
  • Dealer
  • Imbarcazioni
  • Tecnici
  • Dashboard statistiche

Tecnologie

Node.js • Express.js • REST API • MySQL • HTML5 • CSS3 • JavaScript
